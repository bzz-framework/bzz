GETTING STARTED

Kickstart your API fast with ORM support, Authentication and Mock routes. Just install it with pip and you are good to go.

JUST WATCH IT!

ORM-BASED ROUTES

bzz understands the models you've mapped and creates restful routes that you can add to your tornado app.

Learn more › 
import tornado.web
from tornado.ioloop import IOLoop
from mongoengine import *
import bzz

# just create your own documents
class User(Document):
   name = StringField()

routes = bzz.ModelRestHandler. \
            routes_for(User)
application = tornado.web. \
            Application(routes)
application.listen(8888)
IOLoop.instance().start()

MOCKED ROUTES

Specially when working with distributed teams, it's very valuable to provide an API contract early in the process. bzz provides easy mocking of routes to enable the clients of your API to work, while you focus on creating the actual API.

Learn more › 
from tornado.web import Application
from tornado.ioloop import IOLoop
from mongoengine import *
import bzz.mocked_routes as rt

#first create the routes

# status 200 is assumed
get_opt = {
    'body': '{}'
}
post_opt = {
    'body': 'OK'
    'cookies': {
        'super': 'cow'
    },
    'status': 203
}

mocked_routes = rt.MockedRoutes([
  ('GET', '/sample', get_opt),
  ('POST', '/sample', post_opt),
])

application = Application(
    mocked_routes.handlers()
)
application.listen(8888)
IOLoop.instance().start()
