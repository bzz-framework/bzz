ORM-BASED ROUTES

bzz understands the models you've mapped and creates restful routes that you can add to your tornado app.

Learn more â€º
import tornado.web
from tornado.ioloop import IOLoop
from mongoengine import *
import bzz

# just create your own documents
class User(Document):
   name = StringField()

routes = bzz.ModelRestHandler. \
            routes_for(User)
application = tornado.web. \
            Application(routes)
application.listen(8888)
IOLoop.instance().start()
